ignotifier-setbadge

Apr 25, 2015 | Tags:  firefox ,  gmail

Since the Gmail Notifier (ignotifier) addon doesn’t allow us to change its icon badge color (and it cannot be styled via css), I made a simple bash script which allows us to change those colors.

You can the find the script and the documentation on github:

https://github.com/spcmd/ignotifier-setbadge
