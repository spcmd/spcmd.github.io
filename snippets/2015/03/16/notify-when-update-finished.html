Notify when update finished

March 16, 2015 bash

Put this code somewhere in your ~/.bashrc or ~/.bash_aliases

updatefunc() {
    sudo apt-get update ; notify-send -i terminal "Update finished!"
}
upgradefunc() {
    sudo apt-get upgrade ; notify-send -i terminal "Upgrade finished!"
}
alias update=updatefunc
alias upgrade=upgradefunc

Then you can run the update or upgrade command and the system will notify you when itâ€™s finished.
