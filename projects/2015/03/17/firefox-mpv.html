Firefox > mpv

March 17, 2015 bashfirefoxmpvyoutube

Simple bash script for opening youtube videos and other videos in mpv from Firefox & log watched videos, if you want.

What you need

  • Firefox (Download)
  • “Open With” addon (Download)
  • mpv (Download)

Installation

  1. Clone the repository (git clone https://github.com/spcmd/firefox-mpv.git) or download from Github.
  2. Copy firefox-mpv script to somewhere in your $PATH (for example: ~/bin) or wherever you like (but this way you have to edit the firefox-mpv.desktop file – see Notes below!).
  3. Make the script executable (chmod +x /path/to/firefox-mpv)
  4. Copy the firefox-mpv.desktop file to your applications directory (for example: ~/.local/share/applications)
  5. You might have to make the .desktop file executable too (chmod +x /path/to/firefox-mpv.desktop)
  6. Open the firefox-mpv file with a text editor and configure (see Configurable options)
  7. Open the “Open With” addon’s settings in Firefox, click on the “Add” button and browse for the firefox-mpv.desktop file, then click Open.
  8. Now you should have an Open Link with Firefox > mpv in Firefox’s context menu (right click menu).

** Notes: if you didn’t copy the script to somewhere in your $PATH, you have to open the firefox-mpv.desktop file in a text editor and edit the Exec=firefox-mpv line by setting the correct path (Exec=/path/to/firefox-mpv)

Configurable options

These are the default settings!

  • LOGGING="yes" Log watched videos into a html file. (Leave it empty if you don’t want watched videos to be logged: LOGGING="" )
  • DATETIME=$(date "+%Y.%m.%d. %H:%M:%S") Date and time format for the log
  • LOGFILE=~/.firefox-mpv-history.html Log file location (default: your home directory)
  • LOGTEMP=~/.firefox-mpv-history-temp Temporary file location (default: your home directory ; it gets deleted automatically when the logging has finished!)

Why do you need the .desktop file?

The script works without using the .desktop file (in the “Open with” addon’s settings you can use the script instead of the .desktop file), but with the .desktop file, the script works like an application and you have an icon (mpv’s icon) in Firefox’s context menu.
