Firefox > mpv

March 17, 2015 bashfirefoxmpvyoutube

Simple bash script for opening youtube videos and other videos in mpv from Firefox & log watched videos (into a local html file), if you want. Currently the logging option works only with Youtube and Vimeo videos.

Update (2015.04.20.): added auto stop/play Spotify options.

You can find the documentation and the instructions on the Github page:

https://github.com/spcmd/firefox-mpv
